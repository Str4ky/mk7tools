Informations

This is the page where you have to put all of the infos for your character



note

Things that are marked with the (optional) text are not required for a character and can be skipped

Main section​

This is where you put the names for your character, there are 2 names for a character



  • The long one
  • The short one

The long one is the one seen in the character selection screen
The short one is the one seen in the bottom screen while racing


A group is where you can get multiples characters to be next to each other in the character
selection screen if they share the same one



This is where you will put who created this character, so put your name. As stated in the placeholder,
you can put multiples names inside if they are separated by the character : twice



This checkbox lets you enable or disable the Thank You animation that plays once you have completed
the last vanilla cup in grand prix



This is where you can change the standard glider's colors



If you want, you can check this checkbox and it will use the base character's default glider colors



So it will not include the glider color into the CHPACK and will use the base character's default one

If you also want, you can click on this button and it will open a character selection dropdown
where you can set the standard glider's colors from a chosen character

Extra section​

This is where you can set the achievements needed in CTGP-7 to unlock the character



They are 2 type of achievements you can set

  • Achievements level
  • Special achievements

The achievements level is what achievements in the mod you need to have completed in order
to unlock the character and it goes from 1 to 5

Special achievements is what special achievements (blue coins, blue shell, etc...) in the mod
you need to have completed in order to unlock the character and it goes from 0 to 1


This checkbox lets you enable or disable a constant
rotation of the character while racing



A character, when doing a trick over a ramp will get it's
voice played slightly after the animation



This checkbox lets you enable or disable a setting to actually
sync the animation and it's voice when doing a trick over a ramp
